# Introduction et Objectifs du Cours

## Objectifs du cours

Ce cours vise à couvrir l'ensemble du cycle de développement moderne d'applications, depuis les concepts fondamentaux jusqu'au déploiement en production.

---

## Objectifs détaillés

### 1. Compréhension des microservices et des API REST
- Comprendre le concept d'architecture microservices
- Maîtriser les principes des API REST pour la communication entre services

### 2. Développement d'application avec Spring Boot
- Mettre en pratique les concepts théoriques
- Construire une application fonctionnelle avec le framework Spring Boot

### 3. Containerisation standardisée
- Apprendre à packager des applications dans des conteneurs
- Garantir la portabilité et l'uniformité entre différents environnements

### 4. Visualisation du parcours de production
- Comprendre le cycle complet d'une application
- De l'écriture du code à l'exécution sur serveur/container

### 5. Définition des frameworks (Spring Boot)
- Comprendre ce qu'est un framework et son utilité
- Découvrir comment Spring Boot simplifie le développement Java

### 6. Notion de containerisation
- Assimiler le concept de containerisation
- Comprendre son rôle dans le déploiement et la maintenance

### 7. Architecture client-serveur via HTTP
- Étudier l'interaction frontend/backend
- Maîtriser les communications via le protocole HTTP

### 8. Scalabilité horizontale et verticale
- Différencier les deux approches de scalabilité
- Comprendre leurs applications pratiques :  
  - **Scalabilité horizontale** : Ajout de serveurs  
  - **Scalabilité verticale** : Augmentation de la puissance serveur

---

## Compétences visées

À l'issue de ce cours, vous serez capable de :  
- Concevoir et développer une application Spring Boot  
- Créer des API REST pour la communication entre services  
- Containeriser une application avec Docker  
- Comprendre l'architecture complète d'une application moderne  
- Prendre des décisions éclairées sur la scalabilité d'une application
